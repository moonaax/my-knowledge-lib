# 双指针与滑动窗口

## 核心技巧

- 左右指针：排序数组上从两端向中间逼近
- 快慢指针：链表环检测、原地去重
- 滑动窗口：维护一个满足条件的区间，右扩左缩
- 滑动窗口模板：`for right; while 不满足条件 left++; 更新答案`

## 1. 三数之和（#15）

> 找所有和为 0 的不重复三元组

**思路**：排序 + 固定一个数 + 双指针

```java
public List<List<Integer>> threeSum(int[] nums) {
    Arrays.sort(nums);
    List<List<Integer>> res = new ArrayList<>();
    for (int i = 0; i < nums.length - 2; i++) {
        if (nums[i] > 0) break;
        if (i > 0 && nums[i] == nums[i - 1]) continue; // 去重
        int l = i + 1, r = nums.length - 1;
        while (l < r) {
            int sum = nums[i] + nums[l] + nums[r];
            if (sum == 0) {
                res.add(Arrays.asList(nums[i], nums[l], nums[r]));
                while (l < r && nums[l] == nums[l + 1]) l++;
                while (l < r && nums[r] == nums[r - 1]) r--;
                l++; r--;
            } else if (sum < 0) l++;
            else r--;
        }
    }
    return res;
}
```

- 时间 O(n²)，空间 O(1)
- 踩坑：三处去重缺一不可

**📝 解析**

- **思路推导**：暴力三重循环 O(n³)。排序后固定 i，用双指针在 [i+1, n-1] 上找两数之和为 -nums[i]，降到 O(n²)。
- **关键点**：三处去重——① i 和 i-1 相同跳过 ② 找到解后 l 跳过重复 ③ r 跳过重复。`nums[i] > 0` 时直接 break（排序后后面都是正数，不可能和为 0）。
- **易错点**：去重是 `nums[i] == nums[i-1]` 而不是 `nums[i] == nums[i+1]`，后者会漏解。
- **举一反三**：
  - 两数之和（#1）→ 本题的子问题
  - 四数之和（#18）→ 再套一层循环，注意 long 防溢出
  - 最接近的三数之和（#16）→ 不要求等于 0，维护最小差值

## 2. 盛最多水的容器（#11）

> 找两条线使围成的水最多

**思路**：双指针从两端向中间，移动较短的那根

```java
public int maxArea(int[] height) {
    int l = 0, r = height.length - 1, max = 0;
    while (l < r) {
        max = Math.max(max, Math.min(height[l], height[r]) * (r - l));
        if (height[l] < height[r]) l++;
        else r--;
    }
    return max;
}
```

- 时间 O(n)，空间 O(1)
- 为什么移动短板：移动长板面积一定不增

**📝 解析**

- **思路推导**：暴力枚举所有配对 O(n²)。双指针从两端开始，宽度递减，只有移动短板才可能让高度增加从而面积增大。
- **关键点**：移动长板时，高度不可能超过短板，而宽度在减小，所以面积一定不增。
- **易错点**：两端等高时移动哪边都行，不影响最终结果。
- **举一反三**：接雨水（#42）→ 同样的双指针思想，但计算逻辑不同

## 3. 接雨水（#42）

> 计算柱子间能接多少雨水

**思路**：双指针，维护左右最大高度

```java
public int trap(int[] height) {
    int l = 0, r = height.length - 1;
    int lMax = 0, rMax = 0, res = 0;
    while (l < r) {
        lMax = Math.max(lMax, height[l]);
        rMax = Math.max(rMax, height[r]);
        if (lMax < rMax) {
            res += lMax - height[l];
            l++;
        } else {
            res += rMax - height[r];
            r--;
        }
    }
    return res;
}
```

- 时间 O(n)，空间 O(1)
- 核心：每个位置的水量 = min(左最高, 右最高) - 当前高度

**📝 解析**

- **思路推导**：暴力对每个位置找左右最高 O(n²)。预处理左右最大值数组 O(n) 空间。双指针做到 O(1) 空间：较小的那侧的水量是确定的。
- **关键点**：`lMax < rMax` 时处理左边，因为左边的水量由 lMax 决定（右边一定有更高的挡着）。
- **易错点**：先更新 max 再算水量，顺序不能反。
- **举一反三**：单调栈也能解（横向计算），柱状图最大矩形（#84）是类似思路

## 4. 无重复字符的最长子串（#3）

> 找不含重复字符的最长子串长度

**思路**：滑动窗口 + HashSet

```java
public int lengthOfLongestSubstring(String s) {
    Set<Character> set = new HashSet<>();
    int l = 0, max = 0;
    for (int r = 0; r < s.length(); r++) {
        while (set.contains(s.charAt(r))) {
            set.remove(s.charAt(l++));
        }
        set.add(s.charAt(r));
        max = Math.max(max, r - l + 1);
    }
    return max;
}
```

- 时间 O(n)，空间 O(min(n, 字符集大小))

**📝 解析**

- **思路推导**：暴力枚举所有子串 O(n²)。滑动窗口：右指针扩展，遇到重复字符时左指针收缩直到无重复。
- **关键点**：也可用 HashMap 存字符最后出现位置，直接跳 `l = max(l, map.get(c)+1)`，更快。
- **易错点**：用 HashMap 跳跃时必须取 max，因为 l 不能往回走。
- **举一反三**：至多包含两个不同字符的最长子串（#159）→ 窗口条件改为字符种类数

## 5. 找到字符串中所有字母异位词（#438）

> 找 s 中所有 p 的异位词的起始索引

**思路**：固定窗口大小 + 计数数组

```java
public List<Integer> findAnagrams(String s, String p) {
    List<Integer> res = new ArrayList<>();
    if (s.length() < p.length()) return res;
    int[] cnt = new int[26];
    for (char c : p.toCharArray()) cnt[c - 'a']++;
    int[] win = new int[26];
    for (int i = 0; i < s.length(); i++) {
        win[s.charAt(i) - 'a']++;
        if (i >= p.length()) win[s.charAt(i - p.length()) - 'a']--;
        if (Arrays.equals(cnt, win)) res.add(i - p.length() + 1);
    }
    return res;
}
```

- 时间 O(n)，空间 O(1)

**📝 解析**

- **思路推导**：固定窗口大小为 p.length()，滑动时加入右边字符、移除左边字符，比较两个计数数组是否相等。
- **关键点**：`Arrays.equals` 比较 int[26] 是 O(26)=O(1)，所以总时间 O(n)。
- **易错点**：窗口还没满时（`i < p.length()`）不要移除左边字符。
- **举一反三**：字符串的排列（#567）→ 几乎一样，只是返回 boolean

## 6. 最小覆盖子串（#76）

> 找 s 中包含 t 所有字符的最短子串

**思路**：滑动窗口，维护 need 和 have 计数

```java
public String minWindow(String s, String t) {
    int[] need = new int[128], have = new int[128];
    for (char c : t.toCharArray()) need[c]++;
    int required = 0;
    for (int n : need) if (n > 0) required++;

    int formed = 0, l = 0, minLen = Integer.MAX_VALUE, start = 0;
    for (int r = 0; r < s.length(); r++) {
        char c = s.charAt(r);
        have[c]++;
        if (have[c] == need[c]) formed++;
        while (formed == required) {
            if (r - l + 1 < minLen) {
                minLen = r - l + 1;
                start = l;
            }
            char lc = s.charAt(l++);
            if (have[lc] == need[lc]) formed--;
            have[lc]--;
        }
    }
    return minLen == Integer.MAX_VALUE ? "" : s.substring(start, start + minLen);
}
```

- 时间 O(n)，空间 O(1)
- 这是滑动窗口最经典的模板题

**📝 解析**

- **思路推导**：暴力枚举所有子串 O(n²)。滑动窗口：右扩直到覆盖 t 所有字符，然后左缩找最短。用 formed 计数已满足的字符种类数。
- **关键点**：`have[c] == need[c]` 时 formed++，精确判断某个字符刚好满足需求。
- **易错点**：need 数组用 128 大小覆盖 ASCII，不要只用 26（t 可能有大小写）。左缩时先判断再减。
- **举一反三**：这是滑动窗口的万能模板，几乎所有"最短/最小覆盖"问题都能套用

## 7. 两数之和 II（#167）

> 有序数组，找两个数和等于 target

```java
public int[] twoSum(int[] numbers, int target) {
    int l = 0, r = numbers.length - 1;
    while (l < r) {
        int sum = numbers[l] + numbers[r];
        if (sum == target) return new int[]{l + 1, r + 1};
        else if (sum < target) l++;
        else r--;
    }
    return new int[0];
}
```

- 时间 O(n)，空间 O(1)

**📝 解析**

- **思路推导**：有序数组天然适合双指针。和小了左指针右移，和大了右指针左移。
- **关键点**：返回的是 1-indexed，别忘了 +1。
- **易错点**：题目保证有唯一解，不需要处理无解情况。
- **举一反三**：两数之和（#1）→ 无序用哈希表；本题有序用双指针，O(1) 空间

## 8. 长度最小的子数组（#209）

> 找和 ≥ target 的最短连续子数组

```java
public int minSubArrayLen(int target, int[] nums) {
    int l = 0, sum = 0, min = Integer.MAX_VALUE;
    for (int r = 0; r < nums.length; r++) {
        sum += nums[r];
        while (sum >= target) {
            min = Math.min(min, r - l + 1);
            sum -= nums[l++];
        }
    }
    return min == Integer.MAX_VALUE ? 0 : min;
}
```

- 时间 O(n)，空间 O(1)

**📝 解析**

- **思路推导**：全正数数组，和具有单调性，适合滑动窗口。右扩增大和，满足条件后左缩找最短。
- **关键点**：while 不是 if，因为缩一次后可能仍然满足条件，要持续缩。
- **易错点**：数组元素全正是前提，有负数就不能用滑动窗口（和不单调）。
- **举一反三**：和为 K 的子数组（#560）→ 有负数，只能用前缀和+哈希

## 9. 验证回文串（#125）

> 只考虑字母和数字，忽略大小写

```java
public boolean isPalindrome(String s) {
    int l = 0, r = s.length() - 1;
    while (l < r) {
        while (l < r && !Character.isLetterOrDigit(s.charAt(l))) l++;
        while (l < r && !Character.isLetterOrDigit(s.charAt(r))) r--;
        if (Character.toLowerCase(s.charAt(l)) != Character.toLowerCase(s.charAt(r)))
            return false;
        l++; r--;
    }
    return true;
}
```

- 时间 O(n)，空间 O(1)

**📝 解析**

- **思路推导**：双指针从两端向中间，跳过非字母数字字符，忽略大小写比较。
- **关键点**：`Character.isLetterOrDigit` 和 `Character.toLowerCase` 是关键 API。
- **易错点**：空字符串和纯符号字符串应返回 true。跳过字符时注意 `l < r` 边界。
- **举一反三**：验证回文串 II（#680）→ 允许删一个字符，递归检查两种情况

## 10. 颜色分类（#75）

> 原地排序只含 0、1、2 的数组（荷兰国旗问题）

**思路**：三指针

```java
public void sortColors(int[] nums) {
    int l = 0, mid = 0, r = nums.length - 1;
    while (mid <= r) {
        if (nums[mid] == 0) swap(nums, l++, mid++);
        else if (nums[mid] == 1) mid++;
        else swap(nums, mid, r--);
    }
}

private void swap(int[] nums, int i, int j) {
    int tmp = nums[i]; nums[i] = nums[j]; nums[j] = tmp;
}
```

- 时间 O(n)，空间 O(1)，一趟扫描

**📝 解析**

- **思路推导**：计数排序两趟 O(n)。荷兰国旗问题用三指针一趟搞定：l 指向 0 的右边界，r 指向 2 的左边界，mid 遍历。
- **关键点**：遇到 0 和 l 交换后 mid++（交换来的一定是 0 或 1），遇到 2 和 r 交换后 mid 不动（交换来的可能是 0）。
- **易错点**：`mid <= r` 不是 `mid < r`，等号不能少。
- **举一反三**：分类问题的通用思路，快排的 partition 也是类似的三路划分

## 11. 删除有序数组中的重复项（#26）

> 原地删除重复元素，返回新长度

```java
public int removeDuplicates(int[] nums) {
    int slow = 0;
    for (int fast = 1; fast < nums.length; fast++) {
        if (nums[fast] != nums[slow]) {
            nums[++slow] = nums[fast];
        }
    }
    return slow + 1;
}
```

- 时间 O(n)，空间 O(1)

**📝 解析**

- **思路推导**：快慢指针，slow 维护去重后的末尾，fast 遍历。遇到不同的就写入 slow+1。
- **关键点**：返回 `slow + 1` 是新长度，不是下标。
- **易错点**：fast 从 1 开始不是 0，因为第一个元素一定保留。
- **举一反三**：删除有序数组中的重复项 II（#80）→ 允许最多出现两次，改为 `nums[fast] != nums[slow-1]`

## 12. 串联所有单词的子串（#30）

> 找 s 中由 words 所有单词串联形成的子串起始位置

**思路**：滑动窗口，按单词长度分组

```java
public List<Integer> findSubstring(String s, String[] words) {
    List<Integer> res = new ArrayList<>();
    int wordLen = words[0].length(), totalLen = wordLen * words.length;
    Map<String, Integer> need = new HashMap<>();
    for (String w : words) need.merge(w, 1, Integer::sum);

    for (int i = 0; i < wordLen; i++) {
        Map<String, Integer> win = new HashMap<>();
        int l = i, count = 0;
        for (int r = i; r + wordLen <= s.length(); r += wordLen) {
            String w = s.substring(r, r + wordLen);
            win.merge(w, 1, Integer::sum);
            count++;
            while (win.getOrDefault(w, 0) > need.getOrDefault(w, 0)) {
                String lw = s.substring(l, l + wordLen);
                win.merge(lw, -1, Integer::sum);
                l += wordLen;
                count--;
            }
            if (count == words.length) res.add(l);
        }
    }
    return res;
}
```

- 时间 O(n × wordLen)，空间 O(words 数量)

**📝 解析**

- **思路推导**：所有单词等长是关键。按 wordLen 分组（0~wordLen-1 个起点），每组内以单词为单位滑动窗口。
- **关键点**：外层只循环 wordLen 次，内层窗口以 wordLen 为步长滑动，总时间 O(n × wordLen)。
- **易错点**：窗口内某个单词超出 need 的次数时，要从左边缩到不超为止。
- **举一反三**：找到字符串中所有字母异位词（#438）→ 简化版，单个字符为单位的固定窗口
