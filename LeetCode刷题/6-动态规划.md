# 动态规划

## 核心技巧

- DP 五步法：定义状态 → 状态转移方程 → 初始化 → 遍历顺序 → 返回值
- 常见类型：线性 DP、背包、区间 DP、树形 DP、状态压缩
- 空间优化：二维降一维（滚动数组）
- 想不出转移方程就先写暴力递归，加记忆化，再改成递推

## 1. 爬楼梯（#70）

```java
public int climbStairs(int n) {
    if (n <= 2) return n;
    int a = 1, b = 2;
    for (int i = 3; i <= n; i++) {
        int tmp = a + b;
        a = b;
        b = tmp;
    }
    return b;
}
```

- `dp[i] = dp[i-1] + dp[i-2]`，本质是斐波那契
- 时间 O(n)，空间 O(1)

## 2. 打家劫舍（#198）

> 不能偷相邻的房子，求最大金额

```java
public int rob(int[] nums) {
    int prev = 0, curr = 0;
    for (int n : nums) {
        int tmp = Math.max(curr, prev + n);
        prev = curr;
        curr = tmp;
    }
    return curr;
}
```

- `dp[i] = max(dp[i-1], dp[i-2] + nums[i])`
- 时间 O(n)，空间 O(1)

## 3. 最长递增子序列（#300）

```java
public int lengthOfLIS(int[] nums) {
    int[] tails = new int[nums.length];
    int size = 0;
    for (int n : nums) {
        int lo = 0, hi = size;
        while (lo < hi) {
            int mid = (lo + hi) / 2;
            if (tails[mid] < n) lo = mid + 1;
            else hi = mid;
        }
        tails[lo] = n;
        if (lo == size) size++;
    }
    return size;
}
```

- 贪心 + 二分：tails[i] 表示长度为 i+1 的递增子序列的最小末尾
- 时间 O(n·log n)，空间 O(n)
- 普通 DP 是 O(n²)，面试优先写这个

## 4. 零钱兑换（#322）

> 用最少硬币凑出 amount

```java
public int coinChange(int[] coins, int amount) {
    int[] dp = new int[amount + 1];
    Arrays.fill(dp, amount + 1);
    dp[0] = 0;
    for (int i = 1; i <= amount; i++) {
        for (int c : coins) {
            if (c <= i) dp[i] = Math.min(dp[i], dp[i - c] + 1);
        }
    }
    return dp[amount] > amount ? -1 : dp[amount];
}
```

- 完全背包问题
- 时间 O(amount × coins)，空间 O(amount)

## 5. 最长公共子序列（#1143）

```java
public int longestCommonSubsequence(String text1, String text2) {
    int m = text1.length(), n = text2.length();
    int[] dp = new int[n + 1];
    for (int i = 1; i <= m; i++) {
        int prev = 0;
        for (int j = 1; j <= n; j++) {
            int tmp = dp[j];
            if (text1.charAt(i - 1) == text2.charAt(j - 1)) dp[j] = prev + 1;
            else dp[j] = Math.max(dp[j], dp[j - 1]);
            prev = tmp;
        }
    }
    return dp[n];
}
```

- 时间 O(m×n)，空间 O(n)
- 经典二维 DP 压缩到一维

## 6. 编辑距离（#72）

> 把 word1 变成 word2 的最少操作数（插入、删除、替换）

```java
public int minDistance(String word1, String word2) {
    int m = word1.length(), n = word2.length();
    int[] dp = new int[n + 1];
    for (int j = 0; j <= n; j++) dp[j] = j;
    for (int i = 1; i <= m; i++) {
        int prev = dp[0];
        dp[0] = i;
        for (int j = 1; j <= n; j++) {
            int tmp = dp[j];
            if (word1.charAt(i - 1) == word2.charAt(j - 1)) dp[j] = prev;
            else dp[j] = 1 + Math.min(prev, Math.min(dp[j], dp[j - 1]));
            prev = tmp;
        }
    }
    return dp[n];
}
```

- 时间 O(m×n)，空间 O(n)
- 面试高频，必须能写

## 7. 最大子数组和（#53）

```java
public int maxSubArray(int[] nums) {
    int max = nums[0], cur = nums[0];
    for (int i = 1; i < nums.length; i++) {
        cur = Math.max(nums[i], cur + nums[i]);
        max = Math.max(max, cur);
    }
    return max;
}
```

- `dp[i] = max(nums[i], dp[i-1] + nums[i])`
- 时间 O(n)，空间 O(1)
- 也叫 Kadane 算法

## 8. 不同路径（#62）

> 从左上到右下有多少条路径

```java
public int uniquePaths(int m, int n) {
    int[] dp = new int[n];
    Arrays.fill(dp, 1);
    for (int i = 1; i < m; i++)
        for (int j = 1; j < n; j++)
            dp[j] += dp[j - 1];
    return dp[n - 1];
}
```

- 时间 O(m×n)，空间 O(n)

## 9. 最小路径和（#64）

```java
public int minPathSum(int[][] grid) {
    int n = grid[0].length;
    int[] dp = new int[n];
    dp[0] = grid[0][0];
    for (int j = 1; j < n; j++) dp[j] = dp[j - 1] + grid[0][j];
    for (int i = 1; i < grid.length; i++) {
        dp[0] += grid[i][0];
        for (int j = 1; j < n; j++)
            dp[j] = grid[i][j] + Math.min(dp[j], dp[j - 1]);
    }
    return dp[n - 1];
}
```

- 时间 O(m×n)，空间 O(n)

## 10. 单词拆分（#139）

> 判断字符串能否被字典中的单词拆分

```java
public boolean wordBreak(String s, List<String> wordDict) {
    Set<String> set = new HashSet<>(wordDict);
    boolean[] dp = new boolean[s.length() + 1];
    dp[0] = true;
    for (int i = 1; i <= s.length(); i++) {
        for (int j = 0; j < i; j++) {
            if (dp[j] && set.contains(s.substring(j, i))) {
                dp[i] = true;
                break;
            }
        }
    }
    return dp[s.length()];
}
```

- 时间 O(n²)，空间 O(n)

## 11. 最长回文子串（#5）

**思路**：中心扩展

```java
public String longestPalindrome(String s) {
    int start = 0, maxLen = 0;
    for (int i = 0; i < s.length(); i++) {
        int len1 = expand(s, i, i);     // 奇数
        int len2 = expand(s, i, i + 1); // 偶数
        int len = Math.max(len1, len2);
        if (len > maxLen) {
            maxLen = len;
            start = i - (len - 1) / 2;
        }
    }
    return s.substring(start, start + maxLen);
}

private int expand(String s, int l, int r) {
    while (l >= 0 && r < s.length() && s.charAt(l) == s.charAt(r)) { l--; r++; }
    return r - l - 1;
}
```

- 时间 O(n²)，空间 O(1)
- 比 DP 更直观，面试推荐这个写法

## 12. 0-1 背包（模板）

> 每个物品只能选一次，求最大价值

```java
public int knapsack(int[] weights, int[] values, int capacity) {
    int[] dp = new int[capacity + 1];
    for (int i = 0; i < weights.length; i++) {
        for (int j = capacity; j >= weights[i]; j--) { // 逆序！
            dp[j] = Math.max(dp[j], dp[j - weights[i]] + values[i]);
        }
    }
    return dp[capacity];
}
```

- 逆序遍历保证每个物品只用一次
- 完全背包改为正序遍历

## 13. 分割等和子集（#416）

> 能否将数组分成两个和相等的子集

```java
public boolean canPartition(int[] nums) {
    int sum = 0;
    for (int n : nums) sum += n;
    if (sum % 2 != 0) return false;
    int target = sum / 2;
    boolean[] dp = new boolean[target + 1];
    dp[0] = true;
    for (int n : nums) {
        for (int j = target; j >= n; j--) {
            dp[j] = dp[j] || dp[j - n];
        }
    }
    return dp[target];
}
```

- 转化为 0-1 背包：能否凑出 sum/2
- 时间 O(n × sum)，空间 O(sum)

## 14. 最长有效括号（#32）

```java
public int longestValidParentheses(String s) {
    int[] dp = new int[s.length()];
    int max = 0;
    for (int i = 1; i < s.length(); i++) {
        if (s.charAt(i) == ')') {
            if (s.charAt(i - 1) == '(') {
                dp[i] = (i >= 2 ? dp[i - 2] : 0) + 2;
            } else if (i - dp[i - 1] > 0 && s.charAt(i - dp[i - 1] - 1) == '(') {
                dp[i] = dp[i - 1] + 2 + (i - dp[i - 1] >= 2 ? dp[i - dp[i - 1] - 2] : 0);
            }
            max = Math.max(max, dp[i]);
        }
    }
    return max;
}
```

- 时间 O(n)，空间 O(n)

## 15. 买卖股票的最佳时机（#121）

```java
public int maxProfit(int[] prices) {
    int minPrice = Integer.MAX_VALUE, maxProfit = 0;
    for (int p : prices) {
        minPrice = Math.min(minPrice, p);
        maxProfit = Math.max(maxProfit, p - minPrice);
    }
    return maxProfit;
}
```

- 时间 O(n)，空间 O(1)

## 16. 买卖股票的最佳时机 II（#122）

> 可以多次买卖

```java
public int maxProfit(int[] prices) {
    int profit = 0;
    for (int i = 1; i < prices.length; i++) {
        if (prices[i] > prices[i - 1]) profit += prices[i] - prices[i - 1];
    }
    return profit;
}
```

- 贪心：只要涨就卖

## 17. 乘积最大子数组（#152）

```java
public int maxProduct(int[] nums) {
    int max = nums[0], curMax = nums[0], curMin = nums[0];
    for (int i = 1; i < nums.length; i++) {
        if (nums[i] < 0) { int tmp = curMax; curMax = curMin; curMin = tmp; }
        curMax = Math.max(nums[i], curMax * nums[i]);
        curMin = Math.min(nums[i], curMin * nums[i]);
        max = Math.max(max, curMax);
    }
    return max;
}
```

- 同时维护最大和最小（负负得正）
- 时间 O(n)，空间 O(1)

## 18. 正则表达式匹配（#10）

```java
public boolean isMatch(String s, String p) {
    int m = s.length(), n = p.length();
    boolean[][] dp = new boolean[m + 1][n + 1];
    dp[0][0] = true;
    for (int j = 2; j <= n; j++)
        if (p.charAt(j - 1) == '*') dp[0][j] = dp[0][j - 2];
    for (int i = 1; i <= m; i++) {
        for (int j = 1; j <= n; j++) {
            if (p.charAt(j - 1) == '*') {
                dp[i][j] = dp[i][j - 2]; // * 匹配 0 次
                if (p.charAt(j - 2) == '.' || p.charAt(j - 2) == s.charAt(i - 1))
                    dp[i][j] |= dp[i - 1][j]; // * 匹配 1+ 次
            } else if (p.charAt(j - 1) == '.' || p.charAt(j - 1) == s.charAt(i - 1)) {
                dp[i][j] = dp[i - 1][j - 1];
            }
        }
    }
    return dp[m][n];
}
```

- 时间 O(m×n)，空间 O(m×n)

## 19. 不同的二叉搜索树（#96）

```java
public int numTrees(int n) {
    int[] dp = new int[n + 1];
    dp[0] = dp[1] = 1;
    for (int i = 2; i <= n; i++)
        for (int j = 1; j <= i; j++)
            dp[i] += dp[j - 1] * dp[i - j];
    return dp[n];
}
```

- 卡特兰数：`G(n) = Σ G(i-1) × G(n-i)`
- 时间 O(n²)，空间 O(n)

## 20. 解码方法（#91）

> "226" 可以解码为 "BZ"(2,26)、"VF"(22,6)、"BBF"(2,2,6)

```java
public int numDecodings(String s) {
    if (s.charAt(0) == '0') return 0;
    int prev2 = 1, prev1 = 1;
    for (int i = 1; i < s.length(); i++) {
        int curr = 0;
        if (s.charAt(i) != '0') curr = prev1;
        int two = Integer.parseInt(s.substring(i - 1, i + 1));
        if (two >= 10 && two <= 26) curr += prev2;
        prev2 = prev1;
        prev1 = curr;
    }
    return prev1;
}
```

- 时间 O(n)，空间 O(1)
- 踩坑：'0' 不能单独解码
