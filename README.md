## 每篇文章结构


## 使用方式

1. 用 [Obsidian](https://obsidian.md/) 打开本仓库根目录作为 Vault
2. 从 `Android面试题.md` 主索引开始浏览
3. 通过 `[[]]` 双链在文章间跳转
