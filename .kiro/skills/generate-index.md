# Skill: 生成模块索引

## 描述
扫描指定模块目录（或全部模块），自动生成/更新该模块的 README.md 索引文件，保持与实际文档同步。

## 触发条件
用户要求更新索引/生成目录/同步 README 时触发。

## 输入
- `scope`: 可选，指定模块路径（如"Android知识/性能优化"）。不传则扫描所有模块。

## 执行步骤

1. **扫描目录**：遍历目标模块下所有 `.md` 文件（排除 README.md），按文件名排序。

2. **生成 README 内容**：格式与现有 README 保持一致：
   ```markdown
   # {模块名}
   - [[文档1]]
   - [[文档2|文档2（补充说明）]]
   ```
   - 如果文档第一行 `# 标题` 与文件名不同，使用 `[[文件名|标题]]` 别名语法。

3. **对比差异**：与现有 README.md 对比，展示新增/删除的条目。

4. **写入文件**：覆盖写入 `{module}/README.md`。

5. **更新根 README（可选）**：如果模块数量或文档数量发生变化，同步更新根目录 `README.md` 中的统计数字。

## 约束
- 保留现有 README 中手动添加的别名说明（`[[xx|描述]]` 格式）
- 不改变文档排列顺序，如果原 README 有自定义顺序则保留
- 仅处理 `.md` 文件，忽略图片和其他资源
