# 项目知识点分析

## 第一阶段：多轮对话客户端

### 1. 终端版 (`chat.py`)

| 代码 | 知识点 | 对应文档 |
|------|--------|---------|
| `ChatOpenAI(base_url=..., model=...)` | Model I/O — 模型初始化，DeepSeek 兼容 OpenAI 接口 | [[../LangChain专题/2-Model IO详解]] |
| `ChatPromptTemplate.from_messages()` | Prompt Template — 系统提示 + 用户输入模板 | [[../LangChain专题/2-Model IO详解]] |
| `MessagesPlaceholder("history")` | 动态消息占位符 — 运行时插入历史对话 | [[../LangChain专题/2-Model IO详解]] |
| `prompt \| llm` | LCEL 管道符 — 把 prompt 和 llm 串成 Chain | [[../LangChain专题/3-LCEL表达式语言]] |
| `RunnableWithMessageHistory` | Memory 记忆管理 — 自动存取对话历史 | [[../LangChain专题/2-Model IO详解]] |
| `InMemoryChatMessageHistory` | 会话存储 — 内存中保存消息列表 | [[../LangChain专题/2-Model IO详解]] |
| `chain.stream(...)` | 流式输出 — 逐 chunk 返回 | [[../LangChain专题/3-LCEL表达式语言]] |
| `python-dotenv` + `.env` | 环境变量管理 — API Key 不硬编码 | 工程实践 |

### 2. FastAPI 后端 (`server.py`)

| 代码 | 知识点 | 对应文档 |
|------|--------|---------|
| `FastAPI()` + `@app.post("/chat")` | REST API — 把 Chain 暴露为 HTTP 接口 | [[../LangChain专题/7-生产化部署]] |
| `StreamingResponse` + SSE | Server-Sent Events — `data: xxx\n\n` 推送流式数据 | [[../LangChain专题/7-生产化部署]] |
| `chain.astream(...)` | 异步流式调用 — `async for` 配合 FastAPI | [[../LangChain专题/3-LCEL表达式语言]] |
| `CORSMiddleware` | 跨域处理 — Electron 请求 localhost | Web 工程基础 |
| Pydantic `BaseModel` | 请求体校验 — 自动验证 JSON 参数 | FastAPI 基础 |

### 3. Electron 桌面客户端 (`electron/`)

| 文件 | 知识点 | 说明 |
|------|--------|------|
| `main.js` — `BrowserWindow` | Electron 主进程 — 创建窗口、加载 HTML | Electron 基础 |
| `titleBarStyle: 'hiddenInset'` | macOS 原生窗口 — 隐藏标题栏保留红绿灯 | Electron macOS 适配 |
| `package.json` | Node.js 项目管理 — 依赖和启动脚本 | Node.js 基础 |

### 4. 前端界面 (`index.html`)

| 功能 | 知识点 |
|------|--------|
| `fetch()` + `ReadableStream` | Fetch API 流式读取 — 逐块解析 SSE |
| `data-theme` + CSS 变量 | 主题系统 — 深浅色切换，`localStorage` 持久化 |
| `backdrop-filter: blur()` | 毛玻璃效果 |
| `radial-gradient` + `animation` | 环境光晕 — 渐变球体漂移动画 |
| `mask` + `linear-gradient` | 渐变发光边框 — mask-composite 技巧 |
| `@keyframes msgIn` | 消息入场动画 — 淡入 + 上移 |
| `compositionstart/end` | 输入法兼容 — 防止 IME 回车误触发 |
| `text-overflow: ellipsis` | 文本溢出省略 |
| `-webkit-app-region: drag` | Electron 窗口拖拽区域 |

### 5. 工程化

| 操作 | 知识点 |
|------|--------|
| `start.sh` | Shell 脚本 — 后台启动后端 + 前台启动 Electron |
| `.env` / `.env.example` | 密钥管理 — 模板 + gitignore |
| Git 提交管理 | 版本控制 — feat/fix/docs 提交规范 |
