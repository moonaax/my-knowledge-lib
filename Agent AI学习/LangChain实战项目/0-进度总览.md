## 简历呈现建议

项目名改为有业务场景的：**"基于 LangGraph 的技术知识库智能问答系统"**

````
技术栈：LangChain + LangGraph + DeepSeek + FAISS + BM25 + SQLite + FastAPI + Electron + Docker + LangSmith
已实现亮点：
1. LangGraph 状态图编排，手动构建 ReAct 循环（StateGraph + ToolNode + 条件路由）
2. 4 个自定义工具，Agent 自主决策调用
3. SSE 流式输出 + astream_events 细粒度事件推送
4. SQLite 持久化记忆（AsyncSqliteSaver），重启不丢失对话
5. 自纠错循环（工具失败自动重试换策略，扩展 AgentState + corrector 节点）
6. Plan-and-Execute（planner 规划 → executor 执行 → replanner 动态调整）
7. 人机协作节点（interrupt_before，工具调用前暂停等待用户确认/拒绝）
8. RAG 混合检索（向量 + BM25 + RRF 融合排序），评测命中率 72.5%
9. 前端 RetryCard / PlanCard / ToolConfirmCard 组件，实时展示执行过程
10. LangSmith 全链路追踪，LLM/Tool/Graph 执行 trace 可视化
11. Token 成本统计（on_llm_end 事件捕获，每个请求记录 input/output tokens）
12. API Key 认证中间件，开发模式可关闭
13. Docker 容器化（Dockerfile + docker-compose，FAISS 索引 volume 挂载）
````
