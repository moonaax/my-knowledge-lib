## 学习路线

```
第一阶段: 基础入门（1-2 周）
├── 什么是 AI Agent
├── LLM 大语言模型基础
└── Prompt Engineering

第二阶段: 框架学习（2-3 周）
├── LangChain 框架
├── RAG 检索增强生成
└── Function Calling

第三阶段: 进阶设计（2-3 周）
├── Agent 设计模式（ReAct、Plan-and-Execute）
├── Multi-Agent 协作
└── 记忆与状态管理

第四阶段: 实战部署（2-3 周）
├── 完整 Agent 应用开发
├── 部署与性能优化
└── 生产环境最佳实践
```
